    Will ensure that a SprayFire.Controller.Controller is created with the appropriate services added.

    You may use SprayFire.Factory.FireFactory.Factory to set the error handling mechanism used, whether to throw an exception or return a Null Object. By default the type of Null Object returned is SprayFire.Controller.NullObject. You can change the constructor value for the Null Object type given in install_dir/libs/SprayFire/config/SprayFire/environment.php.
