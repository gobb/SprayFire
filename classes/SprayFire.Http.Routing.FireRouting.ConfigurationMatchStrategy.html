    If a $Route is not matched to the request's URI path the default $Route for the bag passed will be used.

    A $Route is considered matched to a request's URI path with a regular expression check. The pattern from the $Route is checked against the string strictly ensuring that the beginning and end of the path matches the pattern exactly.

    getRouteAndParameters(\SprayFire\Http\Routing\RouteBag $Bag, \SprayFire\Http\Request $Request) : array

    Please note that the array returned should have 2 keys: - MatchStrategy::ROUTE_KEY => The Route object matched - MatchStrategy::PARAMETER_KEY => an array of parameters to pass to the action

    It is strongly recommended that you use the constants provided by this interface when setting the keys in the array returned from this method.
