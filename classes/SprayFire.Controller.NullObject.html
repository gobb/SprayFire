    By default this object is returned from the SprayFire.Controller.FireController.Factory implementation and can be used as a 404 object, controlling what is displayed when the appropriate resource could not be found.

    This object does not allow the supporting of services and does not extend SprayFire.Controller.FireController.Base by design. This is intentionally supposed to be as minimal as possible and to perform no operations for the majority of calls.
