    Provides a generic Bootstrapper that will allow applications, plugins and third party libraries to easily integrate their own startup procedures into the framework.

    This class will never implement any functionality not specified in the SprayFire.Object interface. It is explicitly designed in such a way that it should be reasonable for all objects to trace their inheritance back through this object if interfaces they implement require \SprayFire\Object.
