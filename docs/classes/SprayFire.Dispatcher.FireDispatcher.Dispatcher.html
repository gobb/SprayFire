    This class is the primary workhorse of the framework and is responsible for the overall resource processing logic including:

    • Routing the given \SprayFire\Http\Request
    • Initializing the appropriate application based on routing
    • Creation of the appropriate \SprayFire\Controller\Controller and \SprayFire\Responder\Responder objects.
    • Invocation of the Controller and the triggering of an events provided by the default SprayFire install
