There are a variety of options that can be passed to the constructor and log() methods, these options are passed as PHP syslog constants.

The values for constructor options can be found at http://us.php.net/manual/en/function.openlog.php

The values for log() options can be found at http://us.php.net/manual/en/function.syslog.php
